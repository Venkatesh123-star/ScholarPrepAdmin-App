ScholarPrep - Exam Preparation App

Overview
ScholarPrep is an Android-based exam preparation application designed to help students practice and prepare efficiently. 
The app provides a structured platform for conducting exams, managing users, and evaluating performance.
This project includes an **Admin App** for managing exams and student activities.

Features
Student Side
- User Registration & Login
- Attempt Exams/Tests
- View Results & Performance

Admin Side
- Secure Admin Login
- Create and Manage Exams
- Manage Student Accounts
- Monitor Student Performance
- Approve/Control User Access

Tech Stack
- Android Development (Kotlin)
- Firebase Authentication
- Firebase Firestore / Realtime Database
- Android Studio

How It Works
- Students register and log in
- Admin creates exams and controls access
- Students attempt exams
- Results are stored and displayed in real-time

Security Features
- Role-based authentication (Admin & Student)
- Restricted admin access
- Secure Firebase backend

Future Enhancements
- Timer-based exams
- Negative marking system
- Leaderboard & ranking
- Push notifications
- Detailed analytics dashboard
